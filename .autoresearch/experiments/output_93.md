## Experiment Report

**Number:** 93
**Title:** Chat — tab notification badges for unread messages and background agent completion
**Type:** Feature
**Goal:** UI/UX улучшение раздела chat — уведомления о новых сообщениях в неактивных вкладках
**Target:** chat.js, chat-section.js, main.css
**Complexity Impact:** 0 файлов (+3 изменённых)

### Files Modified

- `ui/static/js/modules/chat.js` — `_incrementUnread()`, `_updateDocTitle()`, поля `_unread`/`_agentDone` в объект tab
- `ui/static/templates/chat-section.js` — unread badge, agent done dot, tooltip
- `ui/static/css/main.css` — стили `.tab-unread-badge`, `.tab-label-unread`, `.tab-dot-done`, анимации

### Results

**What was done:**
1. **Unread badge** — пульсирующий фиолетовый бейдж на неактивных вкладках при новых сообщениях от агента
2. **Agent done indicator** — зелёная точка с тройной анимацией мигания когда агент завершил работу в фоне
3. **Tab label highlight** — имя вкладки становится жирным и фиолетовым при непрочитанных
4. **Document title** — заголовок браузера показывает `(N) AutoResearch` при наличии непрочитанных
5. **Tab tooltip** — при наведении показывает "N new messages" или общее число сообщений
6. **Auto-clear** — unread и agentDone сбрасываются при переключении на вкладку

**Working:** yes
**Tests:** skipped — frontend-only (JS + CSS), бэкенд не затронут

### Decision

**Result:** KEEP
**Reason:** Конкретное, полезное улучшение UX чата. Стандартный паттерн IDE/messenger приложений — уведомления на фоновых вкладках. Минимальный объём кода (69 строк), сразу видимый результат.
